# Dr. Nivedha | Pediatric & Lactation Care Website

A clean, easy-to-use one-page website created for Dr. Nivedha's pediatric and lactation clinic at Preethi Hospitals in Madurai. It is designed to be very welcoming for parents and makes it incredibly easy for them to book an appointment.

## 🌟 What This Website Does

This website acts as the online front door for the clinic. It highlights Dr. Nivedha's medical credentials, lists her core services, and builds trust with new patients.

Running Live @ www.lactation-guidance.com

**Key Features:**

- **Mobile-Friendly:** It looks and works perfectly on smartphones, which is how most parents will view it.
- **WhatsApp Booking:** Parents can tap a single button to open a WhatsApp chat directly with the clinic to book a visit.
- **Email Form:** A simple contact form that automatically sends patient messages straight to the clinic's email inbox.
- **Live Patient Reviews:** Automatically shows the clinic's latest 5-star reviews from Google Maps.
- **Location Map:** An interactive Google Map to help parents find the hospital easily.

## 💻 How to View the Website

You don't need any special software to look at this site while working on it.
Simply double-click the `index.html` file, and it will open right up in your regular web browser (like Chrome or Safari).
